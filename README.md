# React-blog-service
 >使用Egg搭建的React-Next-Blog的中台系统。

## 技术栈
    Egg + Mysql

## 项目运行

前端项目地址： https://github.com/zyh940305858/React-Next-Blog

```bash
git clone https://github.com/zyh940305858/React-blog-service
npm i
npm run dev
open http://localhost:7001/
```
